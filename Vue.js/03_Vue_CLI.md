# Vue CLI

<br>

<br>

## Getting started with Vue CLI

<br>

### Installation

```bash
$ npm install -g @vue/cli
```

<br>

### Start project

```bash
$ vue create [PROJECT_NAME]
```

<br>

### Run it

```bash
$ npm run serve

```

<br>

### Install `axios`

```bash
$ npm i axios
```

<br>

### Build

```bash
$ npm build
```
