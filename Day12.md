### Jupyter Notebook alias 하기



> Vim 으로 편집하기
```bash
$ vi .bashrc
```



``i``로 키보드 **편집**들어가기

```vim
alias jn="jupyter notebook"
alias slave="git push origin master git push hub master"
```



``esc`` 3번 치고  (편집모드 빠져나가기)

```vim
:wq
```



> 지금까지 bash에 친 명령어 보기
```bash
$ cat .bash_history
```




> 바꾼 내용 적용 bash에 적용시키기
```bash
$ source ~/.bashrc
```
