## 프로젝트 및 APP 초기 설정



### 프로젝트 만들기

```bash
$ django-admin startproject [project name]
```

<br>

#### `settings.py` 설정

- ALLOWED_HOST = ['*']
- Locale
  - Timezone
  - Language

<br>

###  App 만들기

```bash
$ python manage.py startap [app name]
```

<br>

#### `settings.py` 설정

- INSTALLED_APPS에 app 등록
