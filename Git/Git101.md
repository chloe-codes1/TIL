## Git 주요 명령어

<br>

#### 1. Git 저장소 생성 -  `git init`

```shell
$ git init
```

<br>

#### 2. Git 버전 관리 -  `git status`

- git의 현재 상태가 어떤지 지속적으로 확인해야함!
- *가장 중요한 git 명령어!*

```shell
$ git status
On branch master

# commit 이 아직 없다 (버전 이력이 없다)
No commits yet

# commit 될 변경 사항
Changes to be committed:
  (use "git reset HEAD <file>..." to unstage)


```
